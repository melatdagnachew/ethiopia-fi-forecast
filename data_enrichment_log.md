# Data Enrichment Log

## Overview

This document records the additional data collected and added to the Ethiopia Financial Inclusion unified dataset for Task 1.

The enrichment process aimed to improve the forecasting of Ethiopia's two financial inclusion pillars:

- Access (Account Ownership)
- Usage (Digital Payment Adoption)
